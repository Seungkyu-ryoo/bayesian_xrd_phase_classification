# Bayesian Active Learning for XRD Phase Classification

A Gaussian Process-based active learning framework for efficiently mapping crystallization phase boundaries in Hf‚ÇÅ‚Çã‚ÇìZr‚ÇìO‚ÇÇ thin films under flash lamp annealing conditions.

---

## Overview

Determining the crystalline phase of a material as a function of processing conditions typically requires exhaustive experimentation. This project applies **Bayesian active learning** to dramatically reduce the number of measurements needed by intelligently selecting the most informative next experiment at each step.

The framework operates over a 3-dimensional parameter space:
- **Voltage** (506‚Äì716 V) ‚Äî flash lamp pulse energy
- **Duration** (0.1‚Äì10.1 ms) ‚Äî pulse duration
- **ZrO‚ÇÇ composition** (0‚Äì1) ‚Äî fraction of ZrO‚ÇÇ in Hf‚ÇÅ‚Çã‚ÇìZr‚ÇìO‚ÇÇ

### Two-stage classification

**Stage 1 ‚Äî 2-class (2C):** Classify each point as *amorphous* or *crystalline* using a binary GP classifier. The model iteratively queries points with the highest predictive uncertainty until convergence.

**Stage 2 ‚Äî 3-class (3C):** Within the crystalline region identified by Stage 1, classify the dominant phase as *monoclinic*, *orthorhombic*, or *tetragonal* using a 3-class GP classifier. Acquisition is weighted by both phase uncertainty and the probability of crystallinity, so the model focuses on regions that are most likely to be crystalline *and* most uncertain about the phase.

## Method

### Model
- **Gaussian Process classifier** with Dirichlet likelihood ([GPyTorch](https://gpytorch.ai/))
- **Mat√©rn kernel** (ŒΩ = 0.5) with ARD lengthscales for each input dimension
- Posterior probabilities estimated via Monte Carlo sampling from the GP posterior

### Acquisition function
- **2C:** Maximum predictive entropy over the discrete test grid (480 points)
- **3C:** 4-class entropy ‚Äî joint entropy over {amorphous, monoclinic, orthorhombic, tetragonal} weighted by P(crystalline)

### Stopping criterion
- **2C:** Stops when the mean predictive entropy changes by less than 10% for 4 consecutive iterations
- **3C:** Fixed 50 iterations

Each iteration produces:

**3C stage** ‚Äî Three panels per iteration:
