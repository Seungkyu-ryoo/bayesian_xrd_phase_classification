# Bayesian Active Learning for XRD Phase Classification

A Gaussian Process-based active learning framework for efficiently mapping crystallization phase boundaries in Hf₁₋ₓZrₓO₂ thin films under flash lamp annealing conditions.

---

## Overview

Determining the crystalline phase of a material as a function of processing conditions typically requires exhaustive experimentation. This project applies **Bayesian active learning** to dramatically reduce the number of measurements needed by intelligently selecting the most informative next experiment at each step.

The framework operates over a 3-dimensional parameter space:
- **Voltage** (506–716 V) — flash lamp pulse energy
- **Duration** (0.1–10.1 ms) — pulse duration
- **ZrO₂ composition** (0–1) — fraction of ZrO₂ in Hf₁₋ₓZrₓO₂

### Two-stage classification

**Stage 1 — 2-class (2C):** Classify each point as *amorphous* or *crystalline* using a binary GP classifier. The model iteratively queries points with the highest predictive uncertainty until convergence.

**Stage 2 — 3-class (3C):** Within the crystalline region identified by Stage 1, classify the dominant phase as *monoclinic*, *orthorhombic*, or *tetragonal* using a 3-class GP classifier. Acquisition is weighted by both phase uncertainty and the probability of crystallinity, so the model focuses on regions that are most likely to be crystalline *and* most uncertain about the phase.

---

## Method

### Model
- **Gaussian Process classifier** with Dirichlet likelihood ([GPyTorch](https://gpytorch.ai/))
- **Matérn kernel** (ν = 0.5) with ARD lengthscales for each input dimension
- Posterior probabilities estimated via Monte Carlo sampling from the GP posterior

### Acquisition function
- **2C:** Maximum predictive entropy over the discrete test grid (480 points)
- **3C:** 4-class entropy — joint entropy over {amorphous, monoclinic, orthorhombic, tetragonal} weighted by P(crystalline)

### Stopping criterion
- **2C:** Stops when the mean predictive entropy changes by less than 10% for 4 consecutive iterations
- **3C:** Fixed 50 iterations

## Getting started

### Requirements
```bash
pip install -r requirements.txt
```
Tested with Python 3.11, PyTorch 2.2, GPyTorch 1.12.

### Run
Open `Bayesian_XRD_Phase_Classification.ipynb` and run all cells.  
Figures are saved to `outputs/figures/` and accuracy metrics to `outputs/accuracy/`.

---

## Output visualizations

Each iteration produces:

**2C stage** — P(crystalline) across the voltage–composition space, sliced at 5 pulse durations. The decision boundary (P = 0.5 contour) and the next queried point (★) are overlaid.

**3C stage** — Three panels per iteration:
1. **Phase blend map** — RGB image showing the predicted phase composition within the crystalline region (red = monoclinic, blue = orthorhombic, yellow = tetragonal)
2. **P(crystalline) map** — Probability of crystallization
3. **4-class entropy map** — Acquisition score used to select the next measurement

---

## Data

`ground_truth_phase_fractions.csv` contains XRD phase fractions extracted from Rietveld refinements (GSAS-II) of combinatorial Hf₁₋ₓZrₓO₂ films annealed under varying flash lamp conditions. Each row corresponds to one (voltage, duration, composition) condition.
